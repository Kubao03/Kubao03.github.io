**技术栈**：Solidity · Next.js · Ethers.js · TypeScript · Tailwind CSS

NUS 区块链工程课程项目。将经典狼人杀游戏完全上链，使用 Solidity 智能合约管理所有游戏逻辑与状态。

- 采用 **Commit-Reveal** 方案保证夜间行动保密性
- 支持 Villager / Werewolf / Seer / Witch / Hunter 五种角色
- 前端基于 Next.js + Ethers.js，支持 MetaMask 钱包连接
